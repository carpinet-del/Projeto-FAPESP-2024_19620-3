# Supplier Decision Hybrid Intelligence

## 📌 Project Description
This project aims to develop a hybrid decision-support model for supplier evaluation and development.

It combines:
- Machine Learning
- Explainable AI (SHAP)
- Multicriteria Decision Making (MCDM)
- Group Decision and Consensus (CRP)

## 🎯 Research Questions
- Q1: How can ML + SHAP support supplier evaluation and recommendation?
- Q2: Can ML + CRP automate consensus building?

## 📁 Repository Structure
- `src/` → core code
- `data/` → datasets
- `notebooks/` → experiments
- `docs/` → documentation
